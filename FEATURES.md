# Funcionalidades Sugeridas para o App Pomodoro

## 1. Estatísticas e Histórico ✅
- [x] Contador de pomodoros completados (dia/semana/mês)
- [x] Histórico de sessões
- [x] Gráficos de produtividade
- [x] Estatísticas no menu bar

## 2. Personalização de Tempos
- Configuração de duração de trabalho (padrão: 25 min)
- Configuração de duração de pausa (padrão: 5 min)
- Pausa longa após X pomodoros (ex.: 15-30 min após 4)
- Modo personalizado (tempos customizados)

## 3. Notificações e Alertas
- Notificação quando o timer termina
- Som de alerta configurável
- Notificação visual no menu bar
- Opção de notificação silenciosa

## 4. Persistência e Sincronização
- Salvar progresso entre sessões
- Persistir estatísticas (UserDefaults ou Core Data)
- Sincronização via iCloud (opcional)

## 5. Visual e UX
- Temas (claro/escuro/automático)
- Cores diferentes para trabalho/pausa
- Animações de transição
- Indicador visual de progresso (círculo/barra)
- Tamanho de janela ajustável

## 6. Produtividade
- Lista de tarefas integrada
- Tags/categorias para sessões
- Metas diárias (ex.: 8 pomodoros)
- Streak (dias consecutivos)

## 7. Funcionalidades Avançadas
- Modo foco (bloqueio de distrações)
- Integração com apps (Calendar, Reminders)
- Exportar estatísticas (CSV/JSON)
- Atalhos de teclado customizáveis
- Modo sempre no topo

## 8. Menu Bar Melhorado
- Ícone animado durante execução
- Progresso visual no ícone
- Ações rápidas (pular pausa, adicionar tempo)
