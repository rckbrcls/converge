# Getting Started

## Requirements

- macOS with Xcode installed.
- Swift 5.9 or newer for the package dependency manifest.

## Open the Project

Open `converge.xcodeproj` in Xcode.

## Dependencies

`Package.swift` declares Sparkle as the app update dependency. The package manifest exists for centralized dependency management; the app targets are defined in the Xcode project.

## Running and Testing

Use Xcode schemes for local development, previews, and tests. No command-line build or test command was verified during this documentation pass.
