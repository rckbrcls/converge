# Troubleshooting

## Sparkle updates do not appear

Check `UpdateManager`, Sparkle configuration, release metadata, and the update feed used by the app.

## Notifications do not fire

Review `NotificationManager` and macOS notification permissions.

## Timer state looks wrong

Start with `PomodoroTimer`, `PomodoroSettings`, and session transition logic before changing views.

## Statistics are missing

Check `StatisticsStore` and the session completion path.
